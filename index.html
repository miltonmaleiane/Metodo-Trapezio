Este sistema resolve Integrais usando o método de Trapezio.

  • Use x e y como variáveis. Exemplo: x + y
  • Funções como seno e cosseno devem ser precedidas por Math.. Exemplo: Math.sin(x)
  • Para potência, use **. Exemplo: x**2
